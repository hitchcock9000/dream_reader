# Evaluation protocol

## Quality
Track a small labeled set of dream inputs and review:
- JSON validity rate
- interpretation coherence
- psychological insight usefulness
- actionability of dream guidance

## Latency
Measure:
- time to first analysis response
- time to image availability
- p50 and p95 end-to-end latency by provider

## Cost
Track per-session cost across:
- text generation tokens
- image generation requests
- fallback activation rate

## Reliability
Track:
- provider error rate
- parse failure rate
- speech recognition failure rate
- image generation fallback rate
